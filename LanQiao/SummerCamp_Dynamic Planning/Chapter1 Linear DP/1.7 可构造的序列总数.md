# 1.7 可构造的序列总数

### 问题描述

构造王国一年一度的构造大赛又开始了，这次构造王国的国王将只给定两个数字 $k$ 和 $n$ ，需要大家回答出能构造多少个符合以下条件的序列：

* 序列的长度为 $n$ 。
* $1≤a_1​≤a_2​≤a_3​≤......≤a_n​≤k$。
* $a_i$​ 是 $a_{i−1​}$ 的倍数$（ i≥2 ）$。

由于答案可能非常大，你需要对 $10^9+7$ 取模。

### 输入格式

输入一行包括两个空格分隔的整数 $k$ 和 $n$ 。

数据范围保证：$1≤n,k≤2000$ 。

### 输出格式

输出一个整数表示答案，答案需要对 109+7 取模。

### 样例输入

```text
2 2
```

### 样例输出

```text
3
```

### 说明

样例中符合条件的序列有 $[1,1] ，[1,2]，[2,2]$ 总共三种。

### 运行限制

| 语言      | 最大运行时间 | 最大运行内存 |
| :------ | :----: | :----: |
| C++     |   2s   |  256M  |
| C       |   2s   |  256M  |
| Java    |   3s   |  256M  |
| Python3 |   8s   |  256M  |

### 解题思路

考虑使用动态规划解决问题。

定义 $f[i]f[j]$ 为只考虑前 $i$ 个数且第 $i$ 个数为 $j$ 的合法方案数，初始化时有 $f[1][i]=1(i∈f[1,k])$ 。

因为题意要求 $a_i$​ 得是 $a_{i−1}$​ 的倍数，所以在转移时每个数应该从它的因子转移过来，即：
$$f[i][j] = \sum\limits_{z = 1}^j {[j\bmod z = 0])(f[i - 1][z])}$$
具体操作时，我们可以使用 $O(n​)$ 的复杂度分解 $j$ 的因子来进行转移，也可以提前预处理用枚举倍数的方法得到每个数的因子。最终的答案即为 $\sum\limits_{i = 1}^k {f[n][i]\bmod {{10}^9} + 7}$ 。

由于 $n=2000$ 所以每个数的因子并不多，复杂度可视为 $O(nk)$ 。
